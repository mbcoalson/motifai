## What We Accomplished

### Complete Build: ValidationService with Deterministic Pre-Layer

The core orchestration layer is now done. The system can run end-to-end from a character profile YAML to a final merged ValidationResult.

**Architecture of what was built this session:**

```
Excerpt + CharacterProfile
        │
        ▼
DeterministicValidator          ← free, instant, confidence=1.0
  - Forbidden vocab (regex whole-word, case-insensitive)
  - Signature phrase detection (substring, case-insensitive)
  - Arc stage lookup (by chapter or in-story date)
        │
        │ to_context_injection()
        ▼
LLM Adapter (ClaudeAdapter / OllamaAdapter)
  - Receives scene_context as context=
  - Receives deterministic findings as pre_context=
    (injected under "## Pre-Computed Rule Violations" header
     with "Do not re-flag" instruction)
        │
        ▼
ValidationService.validate()
  - Merges deterministic flags + LLM flags
  - Deduplicates: LLM entry wins when same word flagged by both
  - Recomputes is_valid (any critical flag = False)
        │
        ▼
ValidationResult (merged, final)
```

## Key Bugs Caught and Fixed (Opus Review)

Four issues found during Opus code review after Haiku/Sonnet implemented:

1. **Wiring bug (architectural)**: `pre_context` in adapters was dead code. `ValidationService`
   was combining `scene_context` + deterministic findings into `context=`, so the
   "Do not re-flag" instruction never reached the LLM.
   **Fix**: `ValidationService` now passes `scene_context` as `context=` and
   `deterministic_result.to_context_injection()` as `pre_context=`.

2. **Tautology test**: `assert result.flagged_passages is not None or ... is None` (always True).
   **Fix**: Replaced with meaningful assertion.

3. **Docstring inaccuracy**: `DeterministicValidator` docstring claimed "case-sensitive" matching.
   **Fix**: Corrected to "case-insensitive".

4. **Weak test**: Blank-line separator test only checked for `"\n\n"` in context.
   **Fix**: Now verifies `scene_context` lands in `context=` kwarg and deterministic injection
   lands in `pre_context=` kwarg — testing the actual contract.
